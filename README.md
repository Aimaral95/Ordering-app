Description
  A responsive restaurant ordering web app that allows users to browse a menu, add and remove items from an order, calculate totals in real time, and complete a simulated checkout flow with a confirmation message.

Tech Stack
  - HTML5
  - CSS3
  - JavaScript (Vanilla)

Key Features
  - Dynamic menu rendering from data
  - Real-time order summary with total price calculation
  - Add/remove item functionality using event delegation
  - Modal-based checkout flow
  - Conditional rendering for order and confirmation states
  - Clean, Figma-inspired responsive UI

How to Run
  Install the dependencies and run the project
```
npm install
npm start
```
